# 软件工程个人项目
#### 用命令行运行 BIN/文件夹下的sudoku.jar

#### 生成n个不同终局 (0<n<=1000000)

```
java -jar sudoku.jar -c n
```

#### 求解数独

```
java -jar sudoku.jar -s absolute_path_of_puzzlefile
```

#### 运行GUIBIN/目录下的sudokuGUI.jar，图形界面

```
java -jar sudokuGUI.jar
```
